## 🌊 Works
### 人生クエスト　V5
→ UI/UXを重視した体験設計型タスク管理アプリ

日々の行動を「経験値」として可視化し、継続を習慣化するためのWebアプリです。

小さな行動でも経験値として蓄積され、レベルアップという形で成果が見えることで、
「続けること」自体が楽しくなる仕組みを設計しています。

本アプリでは、クエストの期間ごとの分類（小・中・大）、経験値管理、レベルシステム、
連続達成日数、カテゴリ管理、フィルター機能などを実装し、
継続しやすい体験を意識して改善を重ねています。

単なるタスク管理ではなく、
「行動 → 成長 → 実感」の流れを体験として設計したプロダクトです。
https://taxintian-cloud.github.io/Jinsei-Quest-V1andV2/

## 🎯 Focus

- **UI / UX を最優先にしたWeb制作**
  - 見た目の美しさだけでなく、触っていて心地よい体験を重視
  - アニメーションや演出を通して「世界観」を感じられるUIを設計

- **フロントエンド完結の個人開発**
  - HTML / CSS / JavaScript を中心に、フレームワークに依存しない実装
  - GitHub Pages を活用した公開・運用までを一貫して行います

- **小さく作って、すぐ公開し、改善する**
  - 1作品ごとにテーマを決めて短期間で制作
  - フィードバックや気づきを次の作品に反映する開発スタイル

- **学習と制作を切り離さない**
  - 勉強のためだけのコードではなく「完成させて公開する」ことを重視
  - README や構成を含め、第三者に伝わるアウトプットを意識

- **雰囲気・感情に訴える表現の探求**
  - 深海、カフェ、学習空間など、空気感のあるテーマ設定
  - 技術と表現のバランスを探りながら制作を続けています
 
  - 現在は、制作した作品をベースに収益化・実案件につなげることを目標に活動しています。
